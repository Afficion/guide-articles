## Most Used Git Commands

#### General good practice
- `git status`
- `git log` 

#### Setting up projects
- `git clone`
- `git branch` & `git checkout`
- `git pull`

#### Local Workflow
- What is the local workflow?
  - Branch, make commits, open pull request, collaborate with more commits, and merge
- `git add FILENAME`
- `git commit`
- `git push`
- `git merge`
